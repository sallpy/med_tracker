# Med Tracker API

Backend-приложение для управления медицинскими анализами
## Возможности

-  JWT-аутентификация и авторизация
-  CRUD-операции для медицинских анализов
-  Каждый пользователь видит только свои анализы
-  Автоматическая документация API (Swagger UI, ReDoc)
-  Асинхронная работа с базой данных

## Технологический стек

- **Python** 3.11+
- **FastAPI** - современный веб-фреймворк
- **SQLAlchemy 2.0** - асинхронный ORM
- **SQLite** - легкая файловая база данных (для разработки)
- **Pydantic v2** - валидация данных и сериализация
- **Poetry** - управление зависимостями
- **JWT**-аутентификация

## Быстрый старт

### Предварительные требования

- Python 3.11+
- Poetry (`pip install poetry`)

### 1. Установка


1. Клонируйте репозиторий
   https://github.com/sallpy/med_tracker.git

2. Установите зависимости
   
    `poetry install`

3. Запустите сервер

    `poetry run uvicorn app.main:app --reload`

### 2. Запуск приложения

```bash
poetry run uvicorn app.main:app --reload
```

Приложение будет доступно по адресу: http://localhost:8000

База данных SQLite создается автоматически при первом запуске в файле `bd.sqlite3`.

## Документация API

После запуска приложения доступны интерактивные документации:

- **Swagger UI** (интерактивная документация): http://localhost:8000/docs
- **ReDoc** (альтернативная документация): http://localhost:8000/redoc

## Аутентификация

API использует JWT-аутентификацию для доступа к защищенным эндпоинтам:

- Получите токен через `POST /token` или 🔐'Authorize'
